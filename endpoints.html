💡 EDR Key Concepts

EDR (Endpoint Detection & Response) records every process, network connection, and file event on a host — far more than just AV signatures.

Living-off-the-Land (LotL) attacks use built-in tools (powershell.exe, wmic.exe, certutil.exe) to avoid AV detection. EDR behavioral analysis catches these.

Parent Process Anomaly: Word.exe spawning cmd.exe is suspicious. explorer.exe spawning notepad.exe is normal.
