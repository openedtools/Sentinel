💡 Quarantine

Isolating an endpoint cuts off all network connections — the attacker loses access and malware cannot spread further. This is the fastest way to stop lateral movement. Quarantine first, investigate second.

⚠ Port 4444

Port 4444 is the Metasploit default C2 port — if a machine is communicating outbound on this port, it likely has a reverse shell active. Closing it cuts the attacker's remote access immediately. Port 53 (DNS) is used for covert data exfiltration.

⚖ Collateral Damage

Closing all ports or quarantining the core switch would stop the attack — but would also take down the entire organization. Precision matters. Target only compromised nodes and malicious ports to minimize business disruption.
